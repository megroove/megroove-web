# Megroove

毎日のコーヒーを記録・分析する Web アプリ。

「美味しかったあの一杯を、もう一度淹れられる」をゴールに、記録の手軽さとデータの深さを両立します。

## 機能

- **淹れる記録** — 前回値で自動入力される記録画面。変えたところだけ直すだけで保存完了
- **カフェ記録** — カフェで飲んだドリンクも同様に記録・評価
- **ライブラリ** — 過去の記録一覧（リスト / カード / レコード の3表示モード）
- **分析** — カッピング5軸のレーダーチャートと月間・年間ランキング
- **ストック管理** — 豆・器具・レシピの管理
- **カフェイン管理** — 本日の摂取量と就寝時残留量の予測
- **写真** — 記録ごとに写真を添付（ブリュー・カフェ両対応）
- **PWA 対応** — ホーム画面追加でネイティブアプリのように使用可能

## 使い方

### ホーム画面に追加（推奨）

| iOS Safari | Android Chrome |
|---|---|
| 共有ボタン → 「ホーム画面に追加」 | メニュー → 「ホーム画面に追加」 |

### データについて

全データはブラウザの IndexedDB にローカル保存されます。  
サーバーへの送信は一切ありません。  
設定 → データ管理からエクスポート（JSON）でバックアップできます。

## 技術スタック

- React 19 + TypeScript + Vite
- Tailwind CSS v4
- IndexedDB（`idb` ライブラリ）
- React Router（HashRouter）
- vite-plugin-pwa（Workbox）

## ローカル開発

```bash
npm install
npm run dev
```

## デプロイ

GitHub Pages へのデプロイ:
